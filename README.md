## Técnologias usadas

Front-end:
> Desenvolvido usando: React, Redux, CSS3, HTML5, ES6 e Bootstrap.

## Instalando Dependências

> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação

* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```
